## 指定方法

| 名称                            | 場所                  | 例                                       |
| ------------------------------- | --------------------- | ---------------------------------------- |
| style属性（インラインスタイル） | 各タグのstyle属性     | `<div style="color: red;">Red</div>`     |
| styleタグ                       | head要素内のstyleタグ | `<style>div { color: red; }</style>`     |
| 外部CSS                         | head要素内のlinkタグ  | `<link href="app.css" rel="stylesheet">` |
|                                 | CSS内の `@import`     | `@import 'app.css';`                     |
